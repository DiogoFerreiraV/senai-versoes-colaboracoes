Comandos do Github

    Para acessar a pasta do repositório

  1. cd "nome da pasta"

    2. Setar o usuário

  2. git config --global user.name "nome de usuário"

    3. Setar o email

  3. git config --global user.email "email do usuário"

    4. criar novo repositório

  4. git init

    5. Verificar estado dos arquivos/diretórios

  5. git status

    6. Adicionar todos os arquivos/diretórios

  6. git add .

    7. Comitar um arquivo

  7. git commit -m "mensagem de commit"

    8. Informa a pasta remota

  8. git remote add origin "link da pasta remota"

    9. Publica as alterações no repositório remoto

  9. git push -u origin master

    10. Criando um novo branch

  10. git branch "nome do branch"

    11. Criar um novo branch e trocar

  11. git checkout -b "nome do branch"

    12. Baixar as alterações no repositório remoto

  12. git pull
